## 一、 五個排序法介紹
**1.氣泡排序法:**
氣泡排序是一種基礎的交換排序方式，其運作核心在於重複走訪待排序的數列，每次比較相鄰的兩個元素，若發現前者的數值大於後者則進行交換，使較大的數值如同氣泡般逐步往數列末端移動。這個過程會持續重複直到整份數列完全不需要再進行任何交換為止。

**2.選擇排序法:**
選擇排序的操作邏輯非常直觀，它將數列分為已排序與未排序兩部分，每次從未排序的序列中尋找最小值，並將其與未排序部分的起始位置進行交換。這種方式不論原始數據的排列狀態如何，每一輪都必須掃描剩下的所有元素以確定最小值，雖然效率較低，但其交換次數相對較少。

**3.插入排序法:**
插入排序的原理類似於整理撲克牌的手法，其運作方式是將第一個元素視為已排序，隨後從第二個元素開始逐一取出，並由後向前掃描已排序部分的序列，找到合適的位置後將元素插入。這種演算法在數據量較小或是數列已經接近排序完成的情況下，表現非常優異。

**4.合併排序法:**
快速排序是基於分治法發展出的高效演算法，它會從數列中選取一個基準點（Pivot），並將數列重新排列，使得所有比基準點小的數值移至左側，比基準點大的數值移至右側，最後再透過遞迴的方式對左右兩個子數列進行同樣的處理。由於其處理速度極快，是目前實務應用中普及率最高的排序法之一。

**5.快速排序法:**
合併排序同樣採用分治法，但其策略是先將數列不斷對半切割，直到每個子序列都只剩下一個元素，接著再將這些子序列兩兩合併成一個有序的數列。這種方法的優點在於效能極其穩定，且具備穩定排序的特性，不會改變相同數值元素的原始相對順序。

## 二、 複雜度分析
| 排序法 | 平均時間複雜度 | 空間複雜度 | 穩定性 |
| :--- | :--- | :--- | :--- |
| 冒泡排序 | $O(n^2)$ | $O(1)$ | 穩定 |
| 選擇排序 | $O(n^2)$ | $O(1)$ | 不穩定 |
| 插入排序 | $O(n^2)$ | $O(1)$ | 穩定 |
| 快速排序 | $O(n \log n)$ | $O(\log n)$ | 不穩定 |
| 合併排序 | $O(n \log n)$ | $O(n)$ | 穩定 |

## 四、 結果比較與個人心得
從實驗數據可以明顯觀察到，當數據量 $n$ 增加時，**冒泡排序**的執行時間增加最為劇烈，在 $n=5000$ 時已接近 0.7 秒；相對地，**快速排序**與**合併排序**即使在數據量增加時，執行時間仍維持在 0.01 秒以下，展現了 $O(n \log n)$ 演算法在處理大規模數據時的壓倒性優勢。
